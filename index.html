Intro

Pengertian dan kegunaan

OOP Adalah metode pemrograman terstruktur yang terdiri dari class dan objek yang dapat saling berkomunikasi dan berkirim pesan kepada objek lain.
Kelebihan OOP diantaranya adalah.
Reusable code : code dapat di gunakan kembali suatu saat di butuhkan. Hal ini dapat mempercepat waktu coding dan mempercepat proses nya.
Pengelompokan : dengan adanya class kita dapat memb agi tugas sesuai kebutuhan.
Teamwork : dengan class dan objek yg dapat saling berkomunikasi memudahkan pembuatan program yg di lakukan lebih dari 1 orang. Dengan bekerja dengan team dapat menghemat waktu pembuatan program.
Security : dengan ada nya enkapsulasi, variable tidak dapat di gunakan di sembarang tempat.
